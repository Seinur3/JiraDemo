JiraDemo – Simple Task Management API

This project is a lightweight backend application inspired by Jira.
It demonstrates clean API architecture, separation of concerns, and basic task-management logic built with ASP.NET Core Web API.

The application allows creating projects, adding tasks, commenting on them, and managing data through a structured controller/service architecture.

Features:

Create and manage projects

Create tasks within projects

Update task details and status

Add comments to tasks

Upload and download attachments

Layered architecture (Controllers → Services → DTOs → Models)

Dependency Injection

AutoMapper for DTO mapping

Tech Stack:

ASP.NET Core Web API

Entity Framework Core

AutoMapper

PostgreSQL

JWT Authentication

Swagger UI
